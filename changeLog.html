Piccolo ChangeLog

3.1.2

  • [Modification, pic18] : instruction ldataptr renommée en ldata16ptr ;
  • [Modification, pic18] : section data renommée en data16 ;
  • [Ajout, pic18] : nouvelle section data8, tableau d'octets en Flash ;
  • [Ajout, pic18] : nouvelle instruction ldata8ptr, accès au tableau d'octets en Flash ;

3.1.1

  • [Ajout] : calcul des bornes de la durée d'exécution des routines sans boucle ;
  • [Ajout] : instruction nopbra pour pic18 ;

3.1.0

  • [Ajout] : Ajout d'étiquette pour faciliter la lecture d'instructions structurées ;

3.0.9

  • [Modification] : changement de syntaxe de l'instruction de répétition statique do (N : exp ... exp) instructions end ;
  • [Ajout] : dans un champ de bits, les bits inutilisés consécutifs peuvent être spécifiés par un nombre ;

3.0.8

  • [BUG FIX] 12F629 : le code engendré n'accède plus aux bits STATUS.7 et STATUS.6 ;
  • [BUG FIX] 12F629 : la clé de configuration BOREN est renommée en BODEN ;

3.0.7

  • [Ajout] Calcul de la profondeur de pile utilisée (pic18 uniquement) ;
  • [BUG FIX] pic18 : le code engendré par computed goto était erroné dans les versions précédentes ;
  • [BUG FIX] 18F26K80 : suppression de la clé de configuration MAINTAIN inutile pour ce composant ;

3.0.6

  • [Ajout] ajout de l'instruction fnop pour le pic18 : cette instruction permet d'engendrer des instructions machines dont le code est 0xFxxx, qui sont reconnues comme des NOP ;
  • [Ajout] ajout de l'instruction switch w pour le pic18 ;

3.0.5

  • [BUG FIX] correction de la génération du code de l'instruction computed bra des pic18 : le code engendré était incorrect dans les versions précédentes ;

3.0.4

  • [Ajout] instruction checknobank qui permet de vérifier qu'aucun banc n'est sélectionné (mid-range et pic18);
  • [Ajout] instruction block qui permet d'exprimer un flow de contrôle d'exécution comme un enchaînement de blocs (pic18 uniquement);
  • [BUG FIX] correction de la restitutiuon de W dans les routines d'interruption mid-range ;

3.0.3

  • [Ajout] instruction composée do (n) ... end qui déroule statiquement n fois les instructions incluses;

3.0.2

  • [Ajout, application Cocoa] l'application Cocoa permet d'installer/désinstaller les outils ligne de commande à partir des préférences;
  • [Ajout] le compilateur signale par un warning les variables déclarées par byte qui ne sont pas utilisées;
  • [Ajout] déclaration unused byte qui permet de supprimer le warning provoqué par une variable inutilisée;

3.0.1

  • [BUG FIX, nouveau compilateur pour pic18] correction de la prise en charge de l'instruction ldataptr;
  • [BUG FIX, nouveau compilateur pour pic18] une instruction DECF ou INCF dans un test sans utilité est maintenant signalé comme une erreur;

3.0.0

  • [Ajout] nouveau compilateur pour pic18;

2.0.2

  • [Ajout] calcul des niveaux de pile et des durées d'exécution des routines (pic18 uniquement);
  • [Ajout] construction checkpic (pic18 uniquement);

2.0.1

  • [BUG FIX] program memory pages on baselines;
  • [BUG FIX] mid-range interrupt context saving;

2.0.0

  • Added baseline range support;
