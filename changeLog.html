Piccolo ChangeLog

1.2.1

  • Corrected build bug on Win32;
  • Added ROM_SIZE and BOOTSTRAP_SIZE predefined constants;

1.2.0

  • Corrected various bugs on pic18 bank selection checkings;
  • added support for PIC18 bootloader;
  • changed syntax of bit test conditions;
  • New PIC18 supported device: 18F13K22, 18F14K22, 18F25J10;

1.1.0

  • Add support of mid-ranges pics;
  • Syntax change: 'pic18' keyword is used instead of 'program';
  • Syntax change: 'configuration' keyword is used instead of 'config';
  • Bug fix: hex file for code > 64 kB are now correct;
  • Assembly code generation now generates __config directives;
  • New optimization for "if" instruction;
  • Immediat operands can be compile-time expressions;
  • compile-time constants can now be used;

1.0.0

  • First avalaible release.
